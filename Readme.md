# s3m_parser_es6
s3m praser js for es6.

>这是一个超图s3m开源三维标准格式的cesium加载插件，原项目是通过requirejs加载的，转换成es6后，方便各种项目的加载
。直接cnpm/npm install s3m_parser_es6就可以安装使用了。
